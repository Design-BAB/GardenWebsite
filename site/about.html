What is Garden Corner?

Garden Corner is a personal website where I document my garden progress, share photo updates, and host my Happy Garden game.

The visual design mixes the playful, glossy spirit of old web styles with clean modern layout and accessibility-friendly structure.

Static Hosting First

This project is designed to run without a backend, making it a good fit for Neocities-style hosting.

Accessibility Focus

Semantic landmarks, keyboard-friendly interactions, and descriptive image alt text are included throughout.

Maintainable Code

JavaScript is modular, CSS uses variables, and page logic is split into small focused files.
